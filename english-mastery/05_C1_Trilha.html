Unit 01 / 08 · Emphasis

Inversions — emphatic style.

"Never have I seen such poor monitoring", "Hardly had we deployed when the alarm went off", "Not only did we miss the deadline, but we also broke prod". Inversions adicionam ênfase dramática — comum em RFCs, postmortems formais, apresentações.

TriggerEstruturaExemplo tech
Never / Rarely / Seldom+ aux + sujeitoNever have I seen such poor docs.
Hardly / Scarcely+ had + sujeito + p.p.Hardly had we deployed when issues started.
Not only ... but also+ did/have + sujeitoNot only did we miss SLO, but also lost trust.
No sooner ... than+ had + sujeitoNo sooner had we shipped than the bug appeared.
Only when / Only after+ aux + sujeitoOnly after the postmortem did we understand.
Conditional inversionHad + sujeito + p.p. (= If had)Had we monitored, we would have caught it.

Unit 02 / 08 · Cleft

Cleft sentences — focusing.

"It was the deploy that broke prod" (não outro componente). "What we need is more observability" (foco no objetivo). Cleft sentences destacam um elemento — comum em apresentações e contraposições.

TipoEstruturaExemplo tech
It-cleftIt + be + elemento + that/whoIt was the migration that caused this.
Wh-cleft (What)What + cláusula + be + elementoWhat we need is observability.
All-cleftAll + cláusula + be + elementoAll we need is more time.
Reason cleftThe reason + clause + is/wasThe reason it failed was a config bug.

Unit 04 / 08 · Subjuntivo

Subjunctive — formal recommendations.

Em PT é o "que ele faça". Em EN existe um subjuntivo sutil: depois de "It is essential that...", "I recommend that...", "We demand that..." use a forma BASE do verbo (sem -s, sem to). Comum em RFCs, propostas formais.

TriggerEstruturaExemplo tech
It is essential thatthat + sujeito + base verbIt is essential that he be on the call.
I recommend thatthat + sujeito + base verbI recommend she escalate immediately.
We demand thatthat + sujeito + base verbWe demand that the bug be fixed.
It is critical thatthat + sujeito + base verbIt is critical that we monitor closely.
Wish/If onlywere (subjuntivo)I wish I were senior.

Unit 05 / 08 · Diplomatic

Hedging — diplomatic disagreement.

Brasileiro tende a ser direto demais em code reviews — soa rude. Em EN você suaviza com hedging: "I'm not sure if...", "It might be worth considering...", "I'd argue that...". É a marca do tech leader maduro.

FunçãoHedgeExemplo tech
Suavizar opiniãoI'd argue that... / It seems that...I'd argue that this is over-engineered.
Discordar polidamenteI see your point, but... / I'm not sure I agree...I see your point, but I'm worried about scale.
Sugerir suavementeIt might be worth + ING / Have you considered...?It might be worth using a queue.
Atenuar afirmaçãomight / could / appears to / tends toThis could lead to issues at scale.
Pedir esclarecimentoCould you help me understand...?Could you help me understand the tradeoff?

⚠ Direto demais soa rude em EN

❌ Direto (rude em PR)
This is wrong.
You should change this.
This won't scale.
✅ Hedged (profissional)
I'd argue this might not be quite right.
You might want to consider changing this.
I'm worried this won't scale.

Unit 06 / 08 · Coherence

Discourse markers — coherent writing.

Em RFCs e technical specs, você encadeia argumentos com markers: "Furthermore...", "However...", "Nevertheless...", "That said...", "On balance...". Sem eles, suas docs viram bullet points soltos.

FunçãoMarkerExemplo tech
AdicionarFurthermore · Moreover · In additionFurthermore, latency would increase.
ContrastarHowever · Nevertheless · That saidThat said, microservices add overhead.
Causa/efeitoTherefore · Consequently · As a resultConsequently, we'd need to scale.
ExemplificarFor instance · For example · NamelyFor instance, the auth service.
ConcluirIn conclusion · To sum up · On balanceOn balance, REST is simpler.
HedgingApparently · Allegedly · PresumablyApparently, the issue is intermittent.

Unit 07 / 08 · Collocations

Tech collocations — make / do / take.

"Make a decision" não "do a decision". "Do a code review" não "make a code review". Cada substantivo combina com verbo específico. Brasileiro tropeça constantemente — saber as collocations te coloca anos-luz à frente.
